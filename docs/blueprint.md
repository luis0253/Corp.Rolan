# **App Name**: Corporativo Rolan

## Core Features:

- Vehicle Selection: Display available vehicles with images, capacity, and descriptions for easy selection. Vehicle options will adapt to both light and dark themes.
- Intelligent Route Input: Implement autocomplete for origin and destination fields for address selection and validation.
- Duration Input: Numeric input for specifying the number of service days.
- Pricing Engine: Internal cost calculation incorporating distance, operator costs, and tolls. Display calculated price with breakdown including cost per KM, operator fees, and tolls.
- Calculate CTA: Clear call to action button for triggering the pricing calculation. The button will use the accent color.
- Cost Breakdown Display: Clear display of the cost breakdown including the cost per km, operator fees, tolls, subtotal, tax and the final grand total

## Style Guidelines:

- Primary color: Deep blue (#0A2B4C) for dark backgrounds and text.
- Accent color: Golden (#C4A66A) for buttons, icons, and important elements.
- Background color: White (#FFFFFF) / Bone (#F0F0F0) for light backgrounds and text on dark backgrounds.
- Body font: 'Inter', a grotesque-style sans-serif for body text
- Headline font: 'Space Grotesk', a proportional sans-serif font; using 'Inter' for the body
- The layout is a simple and single-column. Use cards with a balanced, not too wide max width, to keep a focused reading experience
- Incorporate a standardized set of icons
